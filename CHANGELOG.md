# Changelog - Apna Mumbai Local

All notable changes to this project are documented here.

## v3.0 - 2026-02-08 (Glassmorphism Revamp)
- Glassmorphism UI with Mumbai station illustration background
- Blue/teal accent theme with frosted glass elements
- Clickable star ratings replacing slider
- Pill/chip style suggestion buttons
- Simplified review form (removed categories)
- Removed example queries from welcome message
- Fixed top space above chatbot UI
- Fixed URL anchor issue
- Custom subdomain: `apnalocal.streamlit.app`

## v2.5 - 2026-02-07 (Summer Theme & Fixes)
- Aesthetic summer theme with pastel gradients
- Live star rating display on slider
- IST timezone fix for cloud deployment
- Fixed return value mismatch in train lookup
- Removed auto-appended reviews from train responses

## v2.0 - 2026-02-06 (Reviews & Connectivity)
- Google Sheets integration for permanent review storage
- User review submission with ratings
- Multi-platform review scraper
- BEST bus first/last mile connectivity
- Fare calculator for Mumbai local trains
- Time-based train filtering
- Platform info, peak hours, and metro support
- Language support (Hindi/Marathi)

## v1.5 - 2026-02-05 (Real Train Data)
- Scraped real AC train schedules
- Added Western Railway non-AC trains (2,185 trains)
- Mid-route support bringing total to 7,504 trains
- Dynamic suggestions based on user queries
- AC train schedule feature

## v1.0 - Initial Release
- Streamlit chatbot for Mumbai local train info
- Station-to-station train search
- Nearest station guidance for non-station locations
- Smart station name matching
- Basic UI with chat interface
